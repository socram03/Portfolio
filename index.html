Tengo algunos proyectos en un ámbito "general"; estos son:

Obviamente, MeowDB. Una especie de base de datos en donde puedes guardar información en archivos JSON, fácil de usar y hecha en JavaScript con soporte a TypeScript. NPM.

También se hizo una versión de esta librería para Ruby en forma de gema pero no es mantenida.

ReactTemplate, un proyecto en el que se usa React y Express en conjunto a Webpack, en pocas palabras es para aquellos que no quieren usar Create React App o quieren tener todo listo en Webpack con solo las cosas esenciales.

FlutterCalculator, fue mi primera aplicación para Android utilizando Flutter, así como su nombre lo dice, es una calculadora, pero esta cambia en diseño y puedes poner tus propias fotos como fondo o simplemente cambiar el color de un texto.

vscDacezuTheme, mi primer tema de Visual Studio Code, un poco simple y oscuro pero siento que es muy agradable visualmente, intento actualizarlo cada que pueda. Puedes conseguirlo aquí.

vscKeyboardColor, fue una extensión de Visual Studio Code muy simple y quería introducirme en este mundo. Así como su nombre indica, cambia el color de tu teclado en el caso de que hayan advertencias o errores; no está disponible en el Marketplace.

FreeAoi/MP3-Downloader, ¿quieres descargar videos de YouTube como MP3s? Contribuí a esta aplicación de Electron con parte de su diseño actual y moviéndola de JavaScript a TypeScript; está hecha con React.

tttedu304/estreck, es un "centro de código" donde puedes subir y ver trozos de código de personas con tan solo crear una cuenta. Mi aportación abarca una gran parte del código de este.

CraterMaik/myChat, una aplicación web para chatear en tiempo real con usuarios de un servidor de Discord, mi aporte fue relativamente grande al proyecto en general.

OpenInBrowser, su nombre lo dice todo, ¿no es así? Una pequeña extensión de Graviton (editor de código) que abre archivos en el navegador.

Discoruby, una gema de Ruby que no tiene mantenimiento. Su propósito general fue poder hacer bots de Discord con ella, debido a la expansiva API de Discord y con el poco tiempo que contábamos, fue abandonada.
